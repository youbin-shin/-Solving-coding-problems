### 2805. 농작물 수확하기

```python
for t in range(1, int(input()) + 1):
    N = int(input())
    crops = [list(map(int, input())) for _ in range(N)]
    ans = sum(crops[N//2])

    plus = N
    start = 0
    for i in range(N//2-1, -1, -1):
        plus -= 2
        start += 1
        for j in range(plus):
            ans += crops[i][start+j]
            ans += crops[N-i-1][start+j]

    print('#{} {}'.format(t, ans))
```

### 4047. 영준이의 카드 카운팅

                cardnums[1] -= 1

            else:

### 6190. 정곤이의 단조 증가하는 수

```python
for t in range(1, int(input()) + 1):
    N = int(input())
    A = list(map(int, input().split()))
    lst = []

    for i in range(N):
        for j in range(i+1, N):
            num = str(A[i]*A[j])
            for k in range(len(num)-1):
                if int(num[k]) > int(num[k+1]):
                    break
            else:
                lst.append(int(num))
    lst.sort()

    if len(lst) == 0:
        result = -1
    else:
        result = lst[-1]
    print('#{} {}'.format(t, result))
```
